## 项目介绍

智能农业物联网监控平台是一个基于 Web 的农业环境监控与管理系统，实现对农场传感器数据的实时监控、设备管理、告警处理、地块种植管理和产品追溯等功能。

## 快速运行

### 方式一：直接打开（推荐）

1. 克隆/下载项目到本地
2. 进入 `frontend/` 目录
3. 用浏览器（Chrome/Edge 120+）打开 `index.html`

> ⚠️ **注意**：ECharts 图表通过 CDN 引入（`cdn.jsdelivr.net`），需要联网才能正常显示图表。  
> 如需离线使用，请下载 ECharts JS 文件放到 `frontend/js/` 目录并修改 HTML 中的引入路径。

### 方式二：本地HTTP服务

```bash
# Python 3
cd smart-agri-iot/frontend
python -m http.server 8080

# 然后打开浏览器访问
# http://localhost:8080/index.html
```

## 各页面说明

### 🔐 登录页 (`index.html`)

- 4个预定义用户卡片，点击直接登录
- 农业主题渐变背景 + 粒子动画
- 用户信息保存在 `sessionStorage`，全平台共享

### 📊 监控看板 (`dashboard.html`)

- **顶部统计栏**：设备总数、在线数、告警数、批次数
- **6个环境指标卡片**：温度/湿度/土壤水分/CO₂/光照/pH，带趋势箭头和状态标识
- **区域切换**：支持农场A/B/C快速切换，数据同步更新
- **24h趋势折线图**：ECharts 多系列折线图
- **设备在线率环形图**：实时在线/离线/故障比例
- **告警摘要面板**：最新5条告警快览
- **自动刷新**：每5秒模拟数据轻微变化，还原实时感

### 📡 设备管理 (`devices.html`)

- 设备列表：17台设备分页显示（每页8条）
- 多维筛选：设备类型 × 状态 × 农场 × 关键词搜索
- 设备详情弹窗：完整设备信息 + 最近5条传感数据
- 增删改查：演示模式，操作结果实时更新列表

### 🔔 告警中心 (`alerts.html`)

- 告警统计：总数/严重/警告/提示/待处理分类汇总
- 等级分布饼图：ECharts 环形图
- 近14天告警趋势：ECharts 堆叠柱状图
- 告警列表：25条记录，多条件筛选+分页
- 告警处理：确认（填写备注）→ 关闭 完整操作流程

### 🗺️ 地块管理 (`fields.html`)

- 左侧树形导航：农场 > 区域两级结构，点击筛选地块
- 右侧地块卡片：8块地块卡片，包含作物/批次/设备/进度
- 种植批次表格：3个批次的详细信息和进度
- 地块详情弹窗：基本信息 + 当前批次 + 关联设备

### 🔍 追溯档案 (`tracing.html`)

- 批次查询：下拉选择或输入批次号
- 批次基本信息卡片：含进度条
- 种植节点进度图：6个关键节点可视化，已完成节点高亮
- 田间作业时间线：完整追溯链，显示日期/操作人/物料/备注

### 📈 统计报表 (`reports.html`)

- 环境趋势图：近7/14/30天 温度+湿度+CO₂ 折线图
- 告警统计柱状图：按天/周/月 三种聚合粒度
- 设备在线率饼图：按农场分类
- 告警规则表：9条规则完整展示
- 批次统计汇总：面积/设备数/告警次数

## 开发规范

- CSS 变量统一在 `:root` 中定义，主色调 `#2E7D32`
- 所有 JS 文件使用 `"use strict"` 严格模式
- 模拟数据通过 `window.xxx` 暴露为全局变量
- 工具函数（Toast/Badge/用户信息）统一在 `mock-data.js` 末尾定义
