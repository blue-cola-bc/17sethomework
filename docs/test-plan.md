# 智能农业物联网监控平台 — 测试计划

**项目名称**：Smart Agriculture IoT Monitoring Platform

## 一、测试范围

本测试计划覆盖 **M1+M2 里程碑**的可测功能，重点针对前端页面功能验证和数据展示正确性。

| 测试类型 | 覆盖范围 |
|---|---|
| 功能测试 | 所有页面功能流程（7个页面） |
| UI/UX 测试 | 页面布局、样式、响应式 |
| 数据一致性测试 | 模拟数据在各页面间的引用一致性 |
| 边界值测试 | 空数据、极值、筛选条件组合 |

---

## 二、测试环境

| 项目 | 说明 |
|---|---|
| 浏览器 | Chrome 120+、Edge 120+（主测）；Firefox 120+（参考） |
| 分辨率 | 1920×1080（主），1366×768（参考），768×1024（平板参考） |
| 运行方式 | 直接浏览器打开 HTML 文件（file://协议）或本地 HTTP 服务 |
| 依赖 CDN | ECharts 5.4.3（需联网），离线环境需替换为本地文件 |

## 五、缺陷等级定义

| 等级 | 说明 | 举例 |
|---|---|---|
| P0-阻塞 | 功能完全不可用 | 页面无法打开，JavaScript报错导致页面空白 |
| P1-严重 | 主要功能不可用 | 设备列表不显示，图表不渲染 |
| P2-一般 | 功能基本可用但有问题 | 筛选条件不生效，数字计算错误 |
| P3-轻微 | UI/样式问题 | 按钮颜色不统一，间距偏差 |
| P4-建议 | 优化建议 | 动画效果改进，文案优化 |
